Impression | Bootstrap HTML5 Conference and Event Template

Installing Template.

  1. After unzip the download pack, you'll found a Template Folder with all the files.
  2. You can view this Template in any browser, you can display or edit the Template without an internet connection.
  3. Now go to your cpanel or open your FTP Client (like Filezilla) and upload the content of the Template on your server root.
  4. Once the files are done uploading go to www.yourdomainname.com/index.html

CSS Files and Structure

These are the css files that are loaded into templates in Head Section.

Google Web Fonts

By default, the template loads this font from Google Web Font Services, you can change the font with the one that suits you best.

		                
						
						@import url('https://fonts.googleapis.com/css?family=Open+Sans:400,700|Roboto:400,500,700');

Fontface Fonts

By default, the template loads this font from from local for heading texts, you can change the font with the one that suits you best.

You can change fonts from main.css

                       body{
							font-size: 15px;
							font-family: 'Roboto', sans-serif;
							font-weight: 400;
							color: #777777;
							margin: 0;
							line-height: 25px;
							background: #fff;
							-webkit-box-sizing: box-sizing;
							-moz-box-sizing: box-sizing;
							box-sizing: box-sizing;
						}
						h1, h2, h3, h4, h5, h6{	
							font-family: 'Open Sans', sans-serif;
							color: #2d2d2d;
							font-weight: 700;
						}
						p{
							color: #777777;
							font-size: 14px;
							font-family: 'Roboto', sans-serif;
							font-weight: 400;
						}

Configuring Contact Form and Countdown

Contact Form

open form-process.php file from ASSETS->php folder of template and configure mailer function parameters to get it worked

                  
                  $EmailTo = "email@gmail.com";

Countdown

open main.js file from ASSETS->js folder of template, goto line 16 and input event date

Google Map Settings

open jquery.mapit.min.js file from ASSETS->js open conact page goto JS library section and change the latitude/longitude values

						var defaults = {
							latitude: -33.807215,
							longitude: 151.090511,
							zoom: 15,
						};
